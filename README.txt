HOME COIFFURE CI V1
====================
Inclus :
- Catalogue avec les tarifs fournis
- Images fournies par le propriétaire du projet
- Réservation Firebase : email, commune, quartier, prestation, coiffeur, date, heure, adresse, paiement, commentaire
- Choix de coiffeur
- WhatsApp : 07 77 40 59 94
- Formulaire "Devenir coiffeur partenaire"
- Administration des réservations et candidatures
- Recherche et export CSV

À connecter pour une production complète :
- Firebase Authentication pour sécuriser admin.html
- Firebase Storage pour téléversement direct des photos des coiffeurs
- Prestataire marchand pour encaissement Mobile Money réel
- Service d'e-mail/Cloud Functions pour e-mails automatiques
- APK Android : cette V1 web peut servir de base PWA/WebView, mais aucun APK signé n'est inclus.

Déploiement : envoyer tous les fichiers et le dossier images/ à la racine du dépôt GitHub Pages.
